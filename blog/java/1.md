# intelliJ IDEA 编辑器入门

## 快捷键

`cmd + ,`: 打开偏好设置

### 处于 Editor 内

`shift + 回车`: 不中断行, 开启新的下一行

`option + cmd + 回车`: 不中断行, 开启新的上一行

`cmd + 回车`: 拆分一行为两行

`shift + cmd + 回车`: 以比较优雅的姿态结束当前行

`cmd + j`: 提示snippet

`ctrl + .`: 插入一条提示并加入一个 `.`

`cmd + d`: 复制整行

`cmd + 退格`: 删除整行

`右键断点小红点`: 修改断点属性

`shift + shift`: 全局搜索

`cmd + /`: 行级注释

`cmd + option + /`: 块级注释

`shift + cmd + ↑`: 语句上移

`shift + cmd + ↓`: 语句上移

`cmd + f`: 文件内查找
